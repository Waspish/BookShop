# 📚 BookShop

Приложение для поиска книг через Open Library API.  
SPA на чистом JavaScript с роутингом, избранным (localStorage) и страницей деталей.

## ✨ Возможности

- 🔍 Поиск книг по названию/автору (минимум 3 символа)
- ⭐ Добавление в избранное (сохраняется после перезагрузки)
- 📖 Страница деталей книги: описание, авторы, обложка, темы
- 🏠 Показ популярных книг при пустом поиске
- 💾 Избранное синхронизируется между страницами

## 🚀 Быстрый старт

```bash
git clone https://github.com/Waspish/BookShop.git
cd BookShop
npm install
npm run build   # сборка бандла
npm start       # запуск http-server
```

Откройте `http://localhost:8080`

## 🛠 Технологии

- JavaScript (ES2020)
- Rollup (сборка)
- http-server / live-server
- on-change (реактивность состояния)
- Open Library API (бесплатно, без ключа)

## 📦 Команды

| Команда        | Действие                          |
|----------------|-----------------------------------|
| `npm run build` | Сборка в `dist/` (IIFE)          |
| `npm start`     | Запуск http-server на `:8080`    |

## 🔄 Роутинг

| Хеш             | Страница               |
|-----------------|------------------------|
| `#/` или пустой | Поиск книг             |
| `#favorites`    | Избранное              |
| `#book/OL...M`  | Детали книги по ключу  |
